Program

1. 기존 앱 분석

Weakness

  • 단순한 앱에 햄버거 메뉴 사용으로 뎁스 증가
  • 수면 기록이 저장되지 않고 소멸
  • 폰트의 색상과 크기가 일정하여 가독성 저하

strength

  • 타이머 모양을 보고 사용방법을 유추하기 쉬움
  • 꺾은선 그래프를 사용함으로써 수면중 변화를 비교하기 쉬움
  • 수면음과 알람음 볼륨을 따로 설정 가능함

2. 현재 앱의 사용자 의견

  • 10시 반까지 맞춰놨는데 9시에 일어났다든지 이럴 때 캔슬을 누르면 완전히 취소가 되어서 앞에 기록도 사라집니다
  • 잠잔시간과 잠을 푹 자는지 알려줘서 고마운데 한가지 아쉬운건 지나간 기록을 볼수없다는 것입니다
  • 수면 그래프 보여주는 게 제일 흥미로웠어요. 근데 알람 면에서는 잘 일어나지 못하겠어요
  • 다좋은데 맞춰놓은시간보다 일찍깼을때 cancle누르면 그래프기록을 볼 수없어서 그냥 맞춰놓은 시간까지 기다려야하는게 좀..일찍일어났을때도 기록보게 해주세요
  • 수면 사운드 진짜 잠을 잘자게 해주네요.. 요즘 수면 질이 높아져서 개운해요. 샤오미밴드 사용중인데 수면 데이터 활용해서 그래프로 보여주세요.

출처 :구글, 앱스토어 사용자 리뷰, 헬스조선, 연합뉴스, 네이버 블로그

3. 앱 개선 방향

UX Vison

수면음악과 수면패턴을 분석하는 기능을 추가하여 꼬인 수면패턴을 한눈에 확인할 수 있게 하여 불면증을 완화하는데 도움을 준다

As is

  • 자동으로 소멸되는 그래프
  • 가독성이 떨어지는 폰트
  • 불필요한 뎁스의 낭비

To be

  • 그래프 저장으로 생활패턴 파악
  • 메인에서 많은 정보 확인 가능
  • 중요도에 따른 직관적인 디자인
  • 하단 탭바를 배치하여 뎁스 줄이기

Design Concept

사람들이 널리 사용하는 폰트를 사용함으로써 친숙한 느낌을 줌과 동시에 폰트의 굵기를 조절하여 디자인적인 느낌을 살렸음

평소에는 당연하게 생각했던 것들이 디자이너분들의 고심 끝에 나온 결과물이라는 것을 알고는 그분들이 존경스러웠습니다.

이번 프로젝트를 통해 UI/UX에 대한 이해도가 높아졌으며 이로 인해 퍼블리싱을 진행함에 있어서도 도움이 많이 될 것이라고 자부합니다.
